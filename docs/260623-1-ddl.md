# Schema / DDL Extraction Implementation Plan

> **Status: COMPLETE** — All 11 tasks implemented (Jun 23, 2026). All 55 tests pass.
> The `mssqlbak schema` command is fully operational. See the priority order section below for the full implementation checklist.

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `mssqlbak schema` CLI command that reads a `.bak` file and emits a complete migration inventory: `CREATE TABLE` T-SQL (with all constraints, indexes, FK relationships), the definition text of views, procedures, functions, and triggers, a schema/namespace inventory, database principals and roles, object-level permissions, user-defined table types, sequences, and synonyms — everything a migration project needs to understand and recreate the source schema without a live SQL Server.

**Architecture:** A new `mssqlbak/ddl.py` module translates the already-decoded `Table`, `Column`, `Index`, `Constraint`, and `ForeignKey` catalog objects into T-SQL strings. The `recover_catalog_objects` function is extended to attach CHECK/DEFAULT definition text (already available in `sysobjvalues`) to each `Constraint`. A new `recover_module_definitions` function reads module bodies (views, procs, functions, triggers) from the same `sysobjvalues` table. The `schema` command writes a single `schema.sql` file (or one file per table with `--per-table`), with tables emitted in FK-dependency order (parents before children).

**Tech Stack:** Python 3.11+, `typer` (CLI), `mssqlbak.catalog` (existing), `mssqlbak.types` (existing), no new dependencies.

### Bucket 3 — Operational objects

| Object | In user .bak? | Internal storage | Notes |
|--------|:------------:|-----------------|-------|
| Database users | ✅ | `sysowners` status=0x53 (`S`) | ✅ `recover_principals` + `emit_principals_report` (Task 10) |
| Database roles | ✅ | `sysowners` status=0x52 (`R`) | ✅ `recover_principals` + `emit_principals_report` (Task 10) |
| Role memberships | ✅ | `sysprivs` class=1 + `sysowners` | ⚠️ Emitted as comment block; `sp_addrolemember` encoding deferred |
| Object-level permissions | ✅ | `sysprivs` | ✅ `recover_object_permissions` + `emit_principals_report` (Task 10) |
| SQL Agent jobs | ❌ **Not in user .bak** | Stored in `msdb.dbo.sysjobs` | Only in an `msdb` backup; never in a user database backup |
| SSIS packages (SSISDB) | ❌ **Not in user .bak** | Stored in `SSISDB` or `msdb.dbo.sysssispackages` | Separate database; not part of user database backup |
| Linked server definitions | ❌ **Not in user .bak** | Stored in `master.sys.servers` | Not in user database — but linked-server **usage** is detectable as four-part names (`[server].[db].[schema].[obj]`) in module text |
| Server-level logins | ❌ **Not in user .bak** | Stored in `master.sys.syslogins` | Only database-level principals (users/roles) are in user .bak |

> **SQL Agent jobs, SSIS packages, and linked-server definitions cannot be extracted from a user database .bak by design — they live in `msdb` or `master`.** A migration inventory can note their likely existence by scanning module text for known patterns (e.g., four-part linked-server names), but no direct extraction is possible.

## Design decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Output format | Single `schema.sql` by default; `--per-table` flag writes `<schema>.<table>.sql` per table | Single file is paste-able; per-table is useful for selective migration |
| Index emission | Non-clustered indexes emitted as separate `CREATE INDEX` after all `CREATE TABLE` blocks | SQL Server convention; avoids FK order issues |
| Collation | Omit `COLLATE` clauses in this plan | Collation IDs require a lookup table not yet implemented; safe default for migration |
| Views / procs / triggers | Emit raw definition text extracted from `sysobjvalues` with a `GO` separator | Text is already stored verbatim in sysobjvalues; no T-SQL parsing required |
| Objects with no definition text | Emit a `-- [type] [name]: definition not available` comment | Graceful fallback for objects created before the backup |

```python
"""T-SQL DDL generation from mssqlbak catalog objects.

Entry points:
  col_sql_type(col)             → T-SQL type string (e.g. "VARCHAR(50)", "DECIMAL(10,2)")
  emit_create_table(...)        → full CREATE TABLE … statement string
  emit_create_index(...)        → CREATE [UNIQUE] [NONCLUSTERED] INDEX … statement
  topo_sort_tables(tables, fks) → tables ordered so FK parents come before children
  emit_schema(...)              → full schema script (tables + indexes + module defs)
"""
from __future__ import annotations

from mssqlbak import types as t
from mssqlbak.catalog import Column

# Types whose SQL name is fixed (no parameters).
_FIXED: dict[int, str] = {
    t.TINYINT:          "TINYINT",
    t.SMALLINT:         "SMALLINT",
    t.INT:              "INT",
    t.BIGINT:           "BIGINT",
    t.BIT:              "BIT",
    t.MONEY:            "MONEY",
    t.SMALLMONEY:       "SMALLMONEY",
    t.REAL:             "REAL",
    t.FLOAT:            "FLOAT",
    t.DATE:             "DATE",
    t.SMALLDATETIME:    "SMALLDATETIME",
    t.DATETIME:         "DATETIME",
    t.UNIQUEIDENTIFIER: "UNIQUEIDENTIFIER",
    t.ROWVERSION:       "ROWVERSION",
    t.IMAGE:            "IMAGE",
    t.TEXT:             "TEXT",
    t.NTEXT:            "NTEXT",
    t.XML:              "XML",
    t.SQL_VARIANT:      "SQL_VARIANT",
    t.NATIVE_JSON:      "JSON",
}

_CLR_NAMES: dict[int, str] = {
    t.UT_HIERARCHYID: "HIERARCHYID",
    t.UT_GEOMETRY:    "GEOMETRY",
    t.UT_GEOGRAPHY:   "GEOGRAPHY",
}


def col_sql_type(col: Column) -> str:
    """Return the T-SQL type string for *col* (no COLLATE, no nullability)."""
    tid = col.type_id
    if tid in _FIXED:
        return _FIXED[tid]
    if tid == t.CLR_UDT:
        return _CLR_NAMES.get(col.user_type_id, f"/* CLR utype={col.user_type_id} */")
    if tid in (t.DECIMAL, t.NUMERIC):
        label = "DECIMAL" if tid == t.DECIMAL else "NUMERIC"
        return f"{label}({col.precision},{col.scale})"
    if tid == t.DATETIME2:
        return f"DATETIME2({col.scale})"
    if tid == t.DATETIMEOFFSET:
        return f"DATETIMEOFFSET({col.scale})"
    if tid == t.TIME:
        return f"TIME({col.scale})"
    if tid == t.CHAR:
        return f"CHAR({col.max_length})"
    if tid == t.VARCHAR:
        return "VARCHAR(MAX)" if col.max_length == -1 else f"VARCHAR({col.max_length})"
    if tid == t.NCHAR:
        return f"NCHAR({col.max_length // 2})"
    if tid == t.NVARCHAR:
        return "NVARCHAR(MAX)" if col.max_length == -1 else f"NVARCHAR({col.max_length // 2})"
    if tid == t.BINARY:
        return f"BINARY({col.max_length})"
    if tid == t.VARBINARY:
        return "VARBINARY(MAX)" if col.max_length == -1 else f"VARBINARY({col.max_length})"
    return f"/* unknown type_id={tid} */"
```

```python
from __future__ import annotations
from mssqlbak.catalog import (
    Column, Constraint, ForeignKey, Index, Table,
    INDEX_CLUSTERED, INDEX_HEAP, INDEX_NONCLUSTERED,
)


def _q(name: str) -> str:
    """Bracket-quote a SQL identifier."""
    return f"[{name}]"


def _col_def(col: Column, default_constraint_name: str | None,
             default_definition: str | None) -> str:
    """Return a single column definition line (no trailing comma)."""
    parts = [f"    {_q(col.name)} {col_sql_type(col)}"]
    if col.identity_seed is not None:
        parts.append(f"IDENTITY({col.identity_seed},{col.identity_increment})")
    parts.append("NOT NULL" if not col.nullable else "NULL")
    if default_constraint_name and default_definition:
        parts.append(f"CONSTRAINT {_q(default_constraint_name)} DEFAULT {default_definition}")
    return " ".join(parts)


def emit_create_table(
    table: Table,
    indexes: list[Index],
    constraints: list[Constraint],
    foreign_keys: list[ForeignKey],
    colid_to_ordinal: dict[int, int],   # colid → 1-based ordinal in table.columns
    obj_to_table: dict[int, Table],     # object_id → Table (for FK ref names)
) -> str:
    """Return a CREATE TABLE … T-SQL string for *table*.

    *colid_to_ordinal* maps each column's ``colid`` to its 1-based position
    (used to resolve index key columns and FK column references by colid).
    *obj_to_table* maps referenced object_ids to their Table objects so FK
    clauses can emit the correct [schema].[name].
    """
    # Build lookup helpers.
    def_by_col_oid: dict[int, tuple[str, str]] = {}   # default_object_id → (name, def text)
    check_constraints: list[Constraint] = []
    default_constraints: list[Constraint] = []
    for cst in constraints:
        if cst.type_code == "D" and cst.definition:
            def_by_col_oid[cst.object_id] = (cst.name, cst.definition)
        elif cst.type_code == "D":
            def_by_col_oid[cst.object_id] = (cst.name, None)
        elif cst.type_code == "C":
            check_constraints.append(cst)

    ordinal_to_col: dict[int, Column] = {i + 1: c for i, c in enumerate(table.columns)}

    lines: list[str] = []
    lines.append(f"CREATE TABLE {_q(table.schema)}.{_q(table.name)} (")

    # Column definitions
    col_lines: list[str] = []
    for col in table.columns:
        if col.is_column_set or col.graph_type in (2, 5, 8):  # skip synthesised cols
            continue
        dflt_info = def_by_col_oid.get(col.default_object_id) if col.default_object_id else None
        dflt_name, dflt_def = dflt_info if dflt_info else (None, None)
        col_lines.append(_col_def(col, dflt_name, dflt_def))

    # Inline PK / UQ constraints (clustered PK inside CREATE TABLE)
    pk_inline: list[str] = []
    uq_inline: list[str] = []
    for ix in indexes:
        if ix.index_type not in (INDEX_CLUSTERED, INDEX_NONCLUSTERED):
            continue
        if not (ix.is_primary_key or ix.is_unique_constraint):
            continue
        key_cols = ", ".join(
            f"{_q(ordinal_to_col[colid_to_ordinal[c]].name)} ASC"
            for c in ix.key_columns
            if c in colid_to_ordinal and colid_to_ordinal[c] in ordinal_to_col
        )
        if not key_cols:
            continue
        kind = "CLUSTERED" if ix.index_type == INDEX_CLUSTERED else "NONCLUSTERED"
        if ix.is_primary_key:
            pk_inline.append(f"    CONSTRAINT {_q(ix.name)} PRIMARY KEY {kind} ({key_cols})")
        elif ix.is_unique_constraint:
            uq_inline.append(f"    CONSTRAINT {_q(ix.name)} UNIQUE {kind} ({key_cols})")

    # CHECK constraints
    ck_lines = [
        f"    CONSTRAINT {_q(c.name)} CHECK {c.definition}"
        for c in check_constraints
        if c.definition
    ]

    # FK constraints
    fk_lines: list[str] = []
    for fk in foreign_keys:
        ref_tbl = obj_to_table.get(fk.ref_object_id)
        if ref_tbl is None:
            continue
        child_cols = ", ".join(
            _q(ordinal_to_col[colid_to_ordinal[c]].name)
            for c in fk.child_col_ids
            if c in colid_to_ordinal and colid_to_ordinal[c] in ordinal_to_col
        )
        ref_cols = ", ".join(
            _q(col.name)
            for c in fk.ref_col_ids
            for col in [next((col for col in ref_tbl.columns if col.colid == c), None)]
            if col is not None
        )
        if not child_cols or not ref_cols:
            continue
        fk_lines.append(
            f"    CONSTRAINT {_q(fk.name)} FOREIGN KEY ({child_cols}) "
            f"REFERENCES {_q(ref_tbl.schema)}.{_q(ref_tbl.name)} ({ref_cols})"
        )

    all_body = col_lines + pk_inline + uq_inline + ck_lines + fk_lines
    for i, line in enumerate(all_body):
        comma = "," if i < len(all_body) - 1 else ""
        lines.append(line + comma)

    lines.append(");")
    return "\n".join(lines)


def emit_create_index(
    index: Index,
    table: Table,
    colid_to_ordinal: dict[int, int],
) -> str:
    """Return a CREATE INDEX T-SQL string for a non-clustered standalone index."""
    ordinal_to_col: dict[int, Column] = {i + 1: c for i, c in enumerate(table.columns)}
    key_cols = ", ".join(
        f"{_q(ordinal_to_col[colid_to_ordinal[c]].name)} ASC"
        for c in index.key_columns
        if c in colid_to_ordinal and colid_to_ordinal[c] in ordinal_to_col
    )
    unique = "UNIQUE " if index.is_unique else ""
    kind = "CLUSTERED" if index.index_type == INDEX_CLUSTERED else "NONCLUSTERED"
    name = _q(index.name) if index.name else "/* unnamed */"
    return (
        f"CREATE {unique}{kind} INDEX {name} "
        f"ON {_q(table.schema)}.{_q(table.name)} ({key_cols});"
    )
```

`emit_schema` orchestrates the full output: all `CREATE TABLE` + `CREATE INDEX` + module definitions in dependency order.

```python
from mssqlbak.catalog import Schema, CatalogObjects
from mssqlbak.inspect import ObjectInfo


def _colid_to_ordinal(table: Table) -> dict[int, int]:
    """Map each column's colid to its 1-based position in table.columns."""
    return {col.colid: i + 1 for i, col in enumerate(table.columns)}


def _topo_sort(tables: list[Table], fks: list[ForeignKey]) -> list[Table]:
    """Return tables in FK-dependency order (parents before children).

    Tables not participating in any FK are emitted first (stable sort within
    each tier).  Cycles are broken arbitrarily.
    """
    by_oid: dict[int, Table] = {t.object_id: t for t in tables}
    # parent_oids: set of object_ids that are referenced by at least one FK.
    deps: dict[int, set[int]] = {t.object_id: set() for t in tables}
    for fk in fks:
        if fk.parent_object_id in deps and fk.ref_object_id in by_oid:
            deps[fk.parent_object_id].add(fk.ref_object_id)

    order: list[Table] = []
    visited: set[int] = set()

    def _visit(oid: int) -> None:
        if oid in visited:
            return
        visited.add(oid)
        for dep in sorted(deps.get(oid, [])):
            _visit(dep)
        if oid in by_oid:
            order.append(by_oid[oid])

    for tbl in sorted(tables, key=lambda t: t.name):
        _visit(tbl.object_id)
    return order


# Module type codes that have SQL text bodies.
_MODULE_TYPES: frozenset[str] = frozenset({"V", "P", "FN", "TF", "IF", "TR", "RF"})


def emit_schema(
    schema: Schema,
    catalog: CatalogObjects,
    inventory: list[ObjectInfo],
    module_defs: dict[int, str],
) -> str:
    """Return a complete schema script as a single string.

    Sections:
      1. CREATE TABLE statements (FK-dependency order)
      2. CREATE INDEX statements (non-clustered standalone indexes)
      3. Module definitions (views, procs, functions, triggers)
    """
    by_oid: dict[int, Table] = {t.object_id: t for t in schema.tables}
    obj_to_table: dict[int, Table] = by_oid

    sorted_tables = _topo_sort(schema.tables, catalog.foreign_keys)

    parts: list[str] = []
    parts.append("-- Generated by mssqlbak schema\n-- Tables\n")

    for tbl in sorted_tables:
        colid_map = _colid_to_ordinal(tbl)
        idxs = catalog.indexes_for(tbl.object_id)
        csts = catalog.constraints_for(tbl.object_id)
        fks = catalog.fks_for(tbl.object_id)
        parts.append(emit_create_table(tbl, idxs, csts, fks, colid_map, obj_to_table))
        parts.append("GO\n")

    # Non-clustered standalone indexes (not PK, not UQ constraint).
    nci_parts: list[str] = []
    for tbl in sorted_tables:
        colid_map = _colid_to_ordinal(tbl)
        for ix in catalog.indexes_for(tbl.object_id):
            if ix.index_type == INDEX_NONCLUSTERED and not ix.is_primary_key and not ix.is_unique_constraint:
                nci_parts.append(emit_create_index(ix, tbl, colid_map))
    if nci_parts:
        parts.append("\n-- Indexes\n")
        parts.extend(f"{s}\nGO\n" for s in nci_parts)

    # Module definitions.
    inv_by_oid = {o.object_id: o for o in inventory}
    module_parts: list[str] = []
    for obj in sorted(inventory, key=lambda o: o.name):
        if obj.is_system or obj.type_code not in _MODULE_TYPES:
            continue
        defn = module_defs.get(obj.object_id)
        if defn:
            module_parts.append(defn.rstrip())
        else:
            module_parts.append(f"-- {obj.type_label} [{obj.name}]: definition not available")
    if module_parts:
        parts.append("\n-- Modules (views, procedures, functions, triggers)\n")
        parts.extend(f"{s}\nGO\n" for s in module_parts)

    return "\n".join(parts)
```

```python
@app.command()
def schema(
    bak: _BakArg,

    \b
    Without --out: writes schema.sql in the current directory.
    With --out DIR: writes to DIR/schema.sql (or DIR/<schema>.<table>.sql with --per-table).

    Examples:
      mssqlbak schema backup.bak
      mssqlbak schema backup.bak --out /tmp/ddl/

        raise typer.Exit(1) from exc

    out_dir.mkdir(parents=True, exist_ok=True)

    if per_table:
        sorted_tables = _topo_sort(sch.tables, catalog.foreign_keys)
        obj_to_table = {t.object_id: t for t in sch.tables}
        for tbl in sorted_tables:

        typer.secho(
            f"wrote {len(sch.tables)} table files to {out_dir}",
            fg=typer.colors.GREEN,
        )
    else:
        ddl = emit_schema(sch, catalog, inventory, module_defs)
        dest = out_dir / "schema.sql"
```

```python
from mssqlbak.catalog import SchemaInfo

def emit_create_schemas(schemas: list[SchemaInfo], principal_name: dict[int, str]) -> str:
    """Return CREATE SCHEMA statements for non-system schemas."""
    _SYSTEM_SCHEMAS = frozenset({"dbo", "guest", "INFORMATION_SCHEMA", "sys"})
    lines: list[str] = ["-- Schemas\n"]
    for s in sorted(schemas, key=lambda x: x.name):
        if s.name in _SYSTEM_SCHEMAS:
            continue
        owner = principal_name.get(s.owner_principal_id)
        auth = f" AUTHORIZATION {_q(owner)}" if owner else ""
        lines.append(f"CREATE SCHEMA {_q(s.name)}{auth};\nGO\n")
```

## Task 8 — User-defined table types (UDTT)

Table types (`TT` in sysschobjs) are high-risk migration indicators — they have no direct Databricks SQL equivalent. Their column definitions live in `syscolpars` just like regular tables. The goal is to emit `CREATE TYPE … AS TABLE` DDL and flag them explicitly in the migration inventory.

**Files:**
- Modify: `mssqlbak/catalog.py`
- Modify: `mssqlbak/ddl.py`

- [ ] **Step 1: Add `UserTableType` dataclass and `recover_user_table_types` to `catalog.py`**

```python
@dataclass(frozen=True)
class UserTableType:

    schema_name: str
```

```python
def recover_user_table_types(store: PageStore) -> list[UserTableType]:
    """Recover TT objects with their column definitions.

    Uses the same syscolpars→Column pipeline as recover_schema.  Column
    decoding is intentionally simplified: leaf_offset is set to 0 for all
    columns because we only need the type/nullability metadata for DDL
    generation, not physical byte offsets.
    """
    boot = _bootstrap(store)
    schobjs_pg = boot.object_table_first_page(_OBJID_SYSSCHOBJS)
    syscolpars_pg = boot.object_table_first_page(_OBJID_SYSCOLPARS)

    # Collect TT object_ids and names.
    tt_objs: dict[int, tuple[str, int]] = {}   # object_id -> (name, nsid)
    for r in _decode_table(store, schobjs_pg, _SYSSCHOBJS_COLS):
        type_b = r.get("type")
        if type_b is None:
            continue
        if type_b.decode("latin1").strip() != "TT":
            continue
        name_b = r.get("name")
        oid = _u(r.get("id"))
        nsid = _u(r.get("nsid"))
        tt_objs[oid] = (name_b.decode("utf-16-le") if name_b else "", nsid)

    if not tt_objs:
        return []

    # Resolve schema names via the same nsid lookup used in recover_schema.
    # (nsid maps to sysclsobjs class=0x32 entries; this is already done in
    # recover_schema via _bootstrap._schema_names.)
    schema_names = {s.schema_id: s.name for s in recover_schemas(store)}

    # Collect columns per TT object from syscolpars.
    cols_by_obj: dict[int, list] = {}
    for r in _decode_table(store, syscolpars_pg, _SYSCOLPARS_COLS):
        oid = _u(r.get("id"))
        if oid not in tt_objs:
            continue
        number = _u(r.get("number"))
        if number != 0:   # only table columns (number=0)
            continue
        cols_by_obj.setdefault(oid, []).append(r)

    result: list[UserTableType] = []
    for oid, (name, nsid) in sorted(tt_objs.items(), key=lambda x: x[1][0]):
        raw_cols = sorted(cols_by_obj.get(oid, []), key=lambda r: _u(r.get("colid")))
        columns = [
            Column(
                name=r.get("name", b"").decode("utf-16-le") if r.get("name") else f"col{_u(r.get('colid'))}",
                colid=_u(r.get("colid")),
                type_id=_u(r.get("xtype")),
                max_length=_s(r.get("length")),
                precision=_u(r.get("prec")),
                scale=_u(r.get("scale")),
                nullable=bool(_u(r.get("status")) & 0x08),
                leaf_offset=0,
                is_variable=False,
                user_type_id=_u(r.get("utype")),
            )
            for r in raw_cols
        ]
        result.append(UserTableType(
            object_id=oid,
            name=name,
            schema_name=schema_names.get(nsid, "dbo"),
            columns=columns,
        ))
    return result
```

## Task 10 — Database principals, roles, and object permissions

`sysowners` (objid=27) stores all database principals. Empirically confirmed:
- `status=0x52` ('R') → database role (user-defined and built-in)
- `status=0x53` ('S') → SQL user, schema-bound user, or dbo

`sysprivs` (objid=29) stores object-level permissions: `actid` is the SQL Server internal action ID mapped to `GRANT`/`EXECUTE`/`SELECT`/etc.

This task creates a read-only inventory (not executable DDL — that requires SIDs and server-level logins which are in `master`), used for the migration impact report.

**Files:**
- Modify: `mssqlbak/catalog.py`
- Modify: `mssqlbak/ddl.py`

```python
@dataclass(frozen=True)
class Principal:
    """A database user or role from sysowners."""
    principal_id: int
    name: str
    principal_type: str    # 'R' = role, 'S' = user/schema

@dataclass(frozen=True)
class ObjectPermission:
    """One row from sysprivs: a grant of *action* on *object* to *grantee*."""
    grantor_id: int
    grantee_id: int
    object_id: int
    action_id: int
    permission_state: str   # 'GRANT' or 'DENY' (from privattrib bit 0)

# SQL Server internal action IDs for common permissions (non-exhaustive):
_ACTION_NAMES: dict[int, str] = {
    193: "SELECT", 195: "INSERT", 196: "DELETE", 197: "UPDATE",
    224: "EXECUTE", 26: "REFERENCES", 178: "CREATE TABLE",
}

_SYSOWNERS_COLS_LAYOUT = [("id", "int"), ("status", "int"), ("name", "nvarchar")]
_SYSPRIVS_COLS_LAYOUT = [
    ("grantor", "int"), ("grantee", "int"), ("class", "tinyint"),
    ("objid", "int"), ("subid", "int"), ("actid", "int"), ("privattrib", "tinyint"),
]
_OBJID_SYSOWNERS = 27
_OBJID_SYSPRIVS = 29


def recover_principals(store: PageStore) -> list[Principal]:
    """Return all database users and roles from sysowners."""
    boot = _bootstrap(store)
    try:
        pg = boot.object_table_first_page(_OBJID_SYSOWNERS)
        rows = _decode_table(store, pg, _layout(_SYSOWNERS_COLS_LAYOUT))
    except Exception:  # noqa: BLE001
        return []
    return [
        Principal(
            principal_id=_u(r.get("id")),
            name=r.get("name", b"").decode("utf-16-le") if r.get("name") else "",
            principal_type=chr(_u(r.get("status"))),
        )
        for r in rows
    ]


def recover_object_permissions(store: PageStore) -> list[ObjectPermission]:
    """Return object-level permissions from sysprivs."""
    boot = _bootstrap(store)
    try:
        pg = boot.object_table_first_page(_OBJID_SYSPRIVS)
        rows = _decode_table(store, pg, _layout(_SYSPRIVS_COLS_LAYOUT))
    except Exception:  # noqa: BLE001
        return []
    result: list[ObjectPermission] = []
    for r in rows:
        privattrib = _u(r.get("privattrib"))
        state = "DENY" if (privattrib & 0x01) else "GRANT"
        result.append(ObjectPermission(
            grantor_id=_u(r.get("grantor")),
            grantee_id=_u(r.get("grantee")),
            object_id=_u(r.get("objid")),
            action_id=_u(r.get("actid")),
            permission_state=state,
        ))
    return result
```

```python
from mssqlbak.catalog import Principal, ObjectPermission

def emit_principals_report(
    principals: list[Principal],
    permissions: list[ObjectPermission],
    obj_to_name: dict[int, str],
    principal_id_to_name: dict[int, str],
) -> str:
    """Return a comment block summarising database principals and permissions.

    This is an inventory report, not executable DDL: CREATE USER requires
    a server-level login (stored in master, not in this backup).
    """
    lines = ["-- Database Principals (inventory only — CREATE USER requires server login)\n"]
    roles = [p for p in principals if p.principal_type == "R" and p.principal_id >= 16384]
    users = [p for p in principals if p.principal_type == "S" and p.name not in ("dbo","guest","INFORMATION_SCHEMA","sys")]
    user_roles = [p for p in principals if p.principal_type == "R" and p.principal_id < 16384]
    for r in sorted(roles, key=lambda x: x.name):
        lines.append(f"-- CREATE ROLE {_q(r.name)};")
    for u in sorted(users, key=lambda x: x.name):
        lines.append(f"-- CREATE USER {_q(u.name)};  -- requires matching server login")
    for ur in sorted(user_roles, key=lambda x: x.name):
        lines.append(f"-- CREATE ROLE {_q(ur.name)};")

    if permissions:
        lines.append("\n-- Object Permissions\n")
        for perm in permissions:
            action = _ACTION_NAMES.get(perm.action_id, f"action_{perm.action_id}")
            obj = obj_to_name.get(perm.object_id, f"object_{perm.object_id}")
            grantee = principal_id_to_name.get(perm.grantee_id, f"principal_{perm.grantee_id}")
            lines.append(f"-- {perm.permission_state} {action} ON {obj} TO {_q(grantee)};")

    return "\n".join(lines)
```

## Task 11 — External dependency detection (linked-server reference scan)

Linked server definitions live in `master`, not in user database backups. This task scans that text and reports cross-server dependencies as a migration risk flag.

**Files:**
- Modify: `mssqlbak/ddl.py`

- [ ] **Step 1: Add `detect_linked_server_refs` to `ddl.py`**

```python
import re
from mssqlbak.inspect import ObjectInfo

_FOUR_PART_RE = re.compile(
    r'\[([^\]]+)\]\.\[([^\]]+)\]\.\[([^\]]+)\]\.\[([^\]]+)\]'  # [srv].[db].[sc].[obj]
    r'|'
    r'(\w+)\.(\w+)\.(\w+)\.(\w+)'                               # srv.db.sc.obj (unquoted)
)


def detect_linked_server_refs(
    module_defs: dict[int, str],
    inventory: list[ObjectInfo],
    local_db_name: str = "",
) -> list[dict[str, str]]:
    """Return a list of suspected linked-server references found in module text.

    Each entry: {object_name, module_type, server, database, schema, object}
    """
    inv_by_oid = {o.object_id: o for o in inventory}
    results: list[dict[str, str]] = []
    for oid, text in module_defs.items():
        for m in _FOUR_PART_RE.finditer(text):
            server = (m.group(1) or m.group(5) or "").strip()
            database = (m.group(2) or m.group(6) or "").strip()
            # Skip four-part names where the server component is actually a local db
            if local_db_name and server.lower() == local_db_name.lower():
                continue
            obj_info = inv_by_oid.get(oid)
            results.append({
                "in_object": obj_info.name if obj_info else f"oid_{oid}",
                "module_type": obj_info.type_code if obj_info else "?",
                "server": server,
                "database": database,
                "schema": m.group(3) or m.group(7) or "",
                "object": m.group(4) or m.group(8) or "",
            })
    return results
```

## What this plan does NOT cover

### Objects that ARE in a user .bak but deferred to follow-on

| Gap | Reason deferred |
|-----|----------------|
| Collation (`COLLATE` clauses on string columns) | Collation ID → name lookup table not yet implemented |
| Computed column definitions (non-persisted) | Expression text is in `sysobjvalues` — plumbing into `Column` is a follow-on |
| Temporal `PERIOD FOR SYSTEM_TIME` clause | `generated_always_type` on `Column` is available but `FOR SYSTEM_TIME` needs the period-column pair |
| Partition function / scheme full DDL | Names are in `sysclsobjs` (class 0x1e/0x1f); parameter recovery needs additional internal table reads |
| `WITH (DATA_COMPRESSION = …)` on indexes | `Table.compression` is available; trivial one-line follow-on |
| Graph `AS NODE / AS EDGE` clause | `is_node`/`is_edge` on `Table` is available; trivial follow-on |
| Sequence start value / increment | Requires reverse-engineering the sequence-value internal table beyond `sysschobjs` |
| Trigger `FOR / AFTER / INSTEAD OF` metadata | Type modifier stored in a status flag not yet decoded; body text is emitted correctly |
| Role membership (`sp_addrolemember`) | `sysprivs` class encoding for membership grants needs further reverse-engineering |
| CLR assembly binary blob extraction | Name + version available via `sysclsobjs`; assembly binary extraction is a separate project |

### Objects that are NOT in a user database .bak

| Object | Where it lives | Migration note |
|--------|---------------|----------------|
| SQL Agent jobs | `msdb.dbo.sysjobs` | Back up `msdb` and run `mssqlbak` against it |
| SSIS packages | `SSISDB` or `msdb.dbo.sysssispackages` | Separate database backup required |
| Server-level logins | `master.sys.syslogins` | `CREATE USER` DDL emitted as comments only |
| Linked server definitions | `master.sys.servers` | Four-part name usage is detected from module text (Task 11) |
| Database mail / profiles | `msdb` | Not in user .bak |
| Replication metadata | System tables in publication DB + distributor | Out of scope |

These cannot be extracted from a user database backup regardless of implementation effort.
