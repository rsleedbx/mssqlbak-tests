# Design Decisions

One file per decision. Each file captures a question that came up during development,
the options considered, the choice made, and the reasoning.

## Naming

`YYYYMMDD-short-topic.md`

## Template

```markdown
# Title

**Date:** YYYY-MM-DD  
**Status:** accepted

## Question

What was being asked or proposed.

## Options considered

### Option A — …

### Option B — …

## Decision

Which option was chosen.

## Reasoning

Why the chosen option is correct and why the alternatives were rejected.
```
